# CST-LABS

This repository contains **Client-Side Technology (CST) JavaScript Labs** developed as part of the **ITI Training Program**.

The labs focus on strengthening JavaScript fundamentals through hands-on tasks involving:
- User input & output
- Loops & conditions
- String manipulation
- Regular expressions
- Math & Date objects
- DOM output using `document.write`
